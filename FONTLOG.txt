FONTLOG
Nokyung font for the Dai Banna script
========================================

This file provides detailed information on the Nokyung font software.
This information should be distributed along with the Nokyung font and any derivative works.


Basic Font Information
----------------------

Nokyung is a smart Unicode font for Dai Banna.
It uses Graphite for advanced features.
Font sources are published in the repository and a smith open workflow is used for building, testing and releasing.
